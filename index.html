Under the Hood

ML Recommendation Pipeline

Three-version iterative training process using scikit-learn's RandomForestClassifier, improving accuracy with each iteration.

01
Data Collection
520+ quiz sessions stored in SQLite. Each session records scores across 5 domains with timestamps for trend analysis.
SQLite · Python
02
Feature Engineering
14 features per student: raw scores, normalized scores (÷10), best/worst/avg/range — all derived with pandas.
pandas · numpy
03
Model Training
RandomForest with 200 estimators, 80/20 train-test split, 5-fold cross-validation. Best model saved as model.pkl.
scikit-learn · joblib
04
Recommendation
Model predicts strongest course domain from student's feature vector. Output shown in quiz results and PDF report.
RandomForestClassifier

The Process

How It Works

From quiz to recommendation in four steps powered by Python and machine learning.

1

Take the 50-Question Assessment

Answer 10 randomized MCQs in each domain. Options are shuffled every run to prevent pattern memorization.

2

Scores Logged to SQLite Database

Your session is assigned a unique ID and all 5 topic scores are stored with timestamp for historical analysis.

3

ML Model Predicts Your Ideal Course

Our RandomForest model (v1.2, 88.89% accuracy) processes your 14-feature vector and returns the best-fit course domain.

4

Visual Report Generated

A bar chart and PDF report are saved with your scores, recommendation, and topic breakdown for future reference.
